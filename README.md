# student-management-system

## Tech Stack
- MongoDB
- Express.js
- React.js
- Node.js
- Axios

## Features
- Create students
- View students
- Update student details
- Delete students

## Setup

### Backend
cd backend
npm install
npm run dev

### Frontend
cd frontend
npm install
npm start

### Challenges faced
- After creating or editing a student, I had to figure out how to make the list update automatically without refreshing the page

- Deciding how to make the form and student list look clean and readable took some experimenting with CSS.
